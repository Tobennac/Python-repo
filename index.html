Twitter Scraping with Python

In this project, I used Python to scrape tweets from Donald Trump's Twitter account and perform some natural language processing tasks on them. I imported some libraries and modules, such as nltk, spacy, pandas, and numpy, to help me with the data processing and analysis. I also downloaded some nltk data packages, such as punkt and wordnet, which are used for tokenization and lemmatization. Tokenization is the process of splitting a text into smaller units, such as words and sentences. Lemmatization is the process of reducing a word to its base form, such as changing 'running' to 'run'. These steps are useful for normalizing and simplifying the text data. To scrape the tweets, I used a custom function called `scrape_tweets`, which takes a Twitter account name as an input and returns a pandas DataFrame with the tweet text, date, and other metadata. The function uses proxies to avoid being blocked by Twitter and prints some information messages to the standard error stream. The function also handles pagination and stops when it reaches the end of the tweet history or the specified limit. I scraped tweets from Donald Trump's account, which is @realDonaldTrump, and stored the result in a variable called `df`. The DataFrame `df` contains 200 tweets from Donald Trump, along with some information such as the tweet ID, the number of likes, retweets, and replies, and the source device. I used the `head` method to view the first five rows of the DataFrame and the `shape` attribute to check the number of rows and columns. The DataFrame has 200 rows and 10 columns. I also used the `info` method to get a summary of the DataFrame, such as the data types, the number of non-null values, and the memory usage.
